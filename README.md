# APIproject

API CRUD de Usuarios con ExpressJS

Se implementa un CRUD básico de usuarios los métodos HTTP GET, POST, PUT y DELETE.

Además, se implementan middlewares de seguridad que validan el acceso a los endpoints mediante headers personalizados.

Instalación:
Se esta utilizando para esto:
- Node.js
- Express.js
- Git

Clona el repositorio e instala las dependencias
- npm install

Ejecuta el servidor
- node index.js
